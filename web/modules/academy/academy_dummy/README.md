# Academy Dummy module

## Summary

The `academy dummy` module provides dummy content for entity types defined
in the `academy` package. It is installed with the `youvo_development`
profile. This includes:

- Course
- Lecture
- Paragraph
  * Checklist
  * Download
  * Evaluation
  * Excursus
  * Illustration
  * Picture
  * Questionnaire
  * Stats
  * Text
  * Video
- Question
  * Multiple Choice
  * Single Choice
  * Task
  * Textarea
  * Textfield
